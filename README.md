## 365 Diary 

### 소개
- 간편하게 일기를 작성하고 저장할 수 있습니다.
- 키워드로 일기를 검색하여 원하는 기록을 쉽게 찾아볼 수 있습니다.
- 개인정보 보호를 위해 비밀번호 잠금 및 지문 잠금을 설정할 수 있습니다.
